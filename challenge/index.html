Concepts pulled directly from real industrial projects.

🔌
IEC 61131-3 Ladder Diagram
Every challenge uses real LD notation — NO/NC contacts, output coils, TON blocks, and seal-in branches rendered exactly as they appear in Studio 5000 and TIA Portal.
⚡
50 ms PLC Scan Cycle
The simulation runs on a real 50 ms interval — inputs are sampled, logic executes, outputs are written, exactly matching the CPU cycle that runs on every real PLC in service.
🛡
Fail-Safe & Safety Standards
Challenges 2 and 4 teach NC wiring, de-energise-to-trip philosophy, and the IEC 62061 manual-reset requirement — topics every automation engineer must know before going on-site.
🌐
Industrial Networking
Challenge 3 walks through a complete Modbus TCP frame at the byte level — Transaction ID, Protocol ID, Length, Unit ID, Function Code, and data payload — using a live protocol analyser.
⏱
Timers, Counters & Sequencing
TON, TOF, and RTO are the backbone of process timing. Challenge 5 teaches the EN/TT/DN bit model and how an E-Stop mid-cycle correctly resets elapsed time to zero.
🧪
ISA-88 Batch Process Control
The standard used in pharma, food, and chemical plants. Challenge 6 implements the Phase model (Fill → Mix → Drain) with sensor-driven transitions — not open-loop timers.
