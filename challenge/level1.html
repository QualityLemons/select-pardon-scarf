Key Concepts
  • ■ Normally Open (NO) — Contact passes current only when the physical button is held. START is NO.
  • ■ Normally Closed (NC) — Contact passes current at rest; opens when pressed. STOP is NC.
  • ■ Seal-in / Latch — Q0 output feeds back a parallel NO contact so the rung stays energised after START is released.
  • ■ PLC Scan Cycle — Read inputs → Execute logic → Write outputs, repeated every 50 ms.
