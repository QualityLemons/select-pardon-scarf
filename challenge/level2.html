🔵 State Machine Design

This tank uses a 2-setpoint hysteresis controller. The pump starts at the low setpoint (20%) and stays on — latched by Q0's seal-in contact — until the high setpoint (85%) is reached. This prevents the pump from rapidly cycling on and off around a single threshold.

🔴 Fail-Safe Wiring Principle

The high-level sensor (LS_HIGH) is wired as Normally Closed (NC). If the wire breaks, I1 reads 0 — exactly as if the tank were full — so the pump stops. The system fails safely rather than overflowing. This is a fundamental principle of industrial safety design (IEC 60947).

🟠 Sensor Types & I/O Mapping

  • Float switches — mechanical, change state when submerged
  • Physical address I0/I1 maps to input terminals on the PLC hardware
  • Wire break on a NO input = stuck 0 (unsafe start risk)
  • Wire break on an NC input = stuck 0 (safe shutdown)
  • Always specify sensor type (NO/NC) on P&ID drawings
